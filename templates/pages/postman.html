简介

一个帮助开发者快速了解 LeanCloud Rest API，调用 API 的工具。通过 Postman Chrome 扩展实现。
简单介绍下优点：

  • 可以灵活设置 URL，URL Get 参数， HTTP header 参数， JSON 格式数据。
  • 可以把发送的请求转换为 cURL 等语言的实现。
  • 可以通过 Chrome 开发者工具查看实际发出的请求和响应

关于 Postman

Send requests quickly -- 快速发送 HTTP 请求

如何使用

  1. 安装 Postman，Postman 是一个 Chrome 扩展。
  2. 引入 LeanCloud Collection 文件。定义了 LeanCloud RestAPI 调用格式的文件。
  3. 引入 LeanCloud 全局环境变量文件，定义了 host,api 版本的全局变量。
  4. 引入 LeanCloud 应用级别环境变量文件，定义了应用的 AppID，AppKey，Master Key。可以替换成开发者自己的应用的相关值
